For compiling the software, VLC is needed:
sudo apt install vlc
sudo apt-get install qt5-default
sudo apt install libvlc-dev

Also the libvlc libraries are needed and Qt5 libraries

For cross plattform development on linux use:
git clone https://github.com/mxe/mxe.git
cd mxe && make qt5

This will take a long time!

Add mxe to path:
export PATH="<mxe root>/usr/bin:$PATH"

Get to the directory of your app, and run the Qt Makefile generator tool:
<mxe root>/usr/bin/i686-w64-mingw32.static-qmake-qt5

Build your project with:
make

There should be a binary in the ./release directory for windows

mxe needs requirements:
sudo apt-get install  autoconf automake autopoint bash bison bzip2 cmake flex gettext git g++ gperf intltool libffi-dev libtool libltdl-dev libssl-dev libxml-parser-perl make openssl patch perl pkg-config python ruby scons sed unzip wget xz-utils
